CodeSystem: Unidade de Medida

Official URL: http://www.saude.gov.br/fhir/r4/CodeSystem/BRUnidadeMedida Version: 0.1.0
Standards status: Informative Maturity Level: 1 Computable Name: BRUnidadeMedida

Code System utilizado para definir a unidade de medida de um medicamento prescrito, para consumo ou especificação do fabricante.

This Code system is referenced in the content logical definition of the following value sets:

  • BRUnidadeConsumo
  • BRUnidadeMedidaMedicamento

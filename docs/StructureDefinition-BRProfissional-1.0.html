Resource Profile: Profissional

URL oficial: http://www.saude.gov.br/fhir/r4/StructureDefinition/BRProfissional-1.0 Version: 0.1.0
Status do padrão: Informative Nível de maturidade do FHIR (FMM): 1 Nome computável: BRProfissional

Indivíduo que realiza atividades profissionais.

O profissional de saúde é sujeito ativo para os processos informativos de dados em saúde, por isso é necessário a criação de um profile para esse recurso, que se adpate à realidade brasileira.

Usage:

  • Refer to this Resource Profile: Contato Assistencial, Lotação Profissional and Prescrição de Medicamento

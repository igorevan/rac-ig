Extension: Intervalo de Doses

Official URL: http://www.saude.gov.br/fhir/r4/StructureDefinition/BRIntervaloDoses Version: 1.0.0
Standards status: Informative Maturity Level: 1 Computable Name: BRIntervaloDoses

Extensão para descrever uma unidade de tempo referenciada pelo UCUM.

Context of Use

    Esta extensão pode ser usada nos seguintes elementos

    • Element ID: MedicationRequest

Usage info

Usos:

  • Usa este Extensão: Prescrição de Medicamento
  • Exemplos para este Extensão: Bundle/bundle-example-rac-1 and Bundle/bundle-example-rac-2

You can also check for usages in the FHIR IG Statistics
